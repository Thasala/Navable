# Navable Privacy Policy

Effective date: May 18, 2026

Navable is a Chrome extension designed to help users navigate webpages using voice or typed commands, page structure scanning, screen-reader-friendly feedback, and optional AI-assisted summaries and answers.

## Data Navable processes

Navable may process webpage content and page structure, such as headings, links, buttons, form labels, landmarks, and visible text, in order to provide navigation, summaries, and page-related answers.

Navable may also process user commands, including typed commands and spoken commands, when the user chooses to use voice features.

## How data is used

Data is used only to provide Navable's core features, including webpage navigation, accessibility feedback, page summaries, transcription, translation, and question answering.

Navable does not sell user data.

Navable does not use user data for advertising, creditworthiness, lending, or unrelated purposes.

## AI and backend features

Some optional AI-assisted features may send a minimized page summary and the user's command to the Navable backend API. The backend may use an AI service to generate summaries, answers, translations, or transcriptions.

Sensitive fields such as passwords and payment fields are excluded from summaries by design where supported.

## Local storage

Navable may store extension preferences locally or in Chrome sync storage, such as language, output mode, AI mode, continuous listening, and privacy settings.

## Remote code

Navable does not load or execute remotely hosted JavaScript or WebAssembly. All extension code is packaged with the extension. Backend communication is used only for data responses such as summaries, transcription, translation, and answers.

## Data sharing

Navable does not sell user data. Data may be sent to backend or AI services only when needed to provide requested extension features.
